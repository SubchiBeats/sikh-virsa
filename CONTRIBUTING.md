# Contributing

Corrections are welcome. Accuracy matters more than adding volume.

## Content corrections

A useful correction includes:

1. the exact claim or field to change;
2. the proposed replacement;
3. the strongest available source;
4. whether the source establishes history, records Sikh tradition, or offers an
   interpretation; and
5. any known disagreement among sources or traditions.

For Gurbani, include the exact ang and check the Gurmukhi in context. Do not
label a quotation as Sri Guru Granth Sahib Ji solely because it is popularly
attributed to a Guru or Bhagat.

For historical material, preserve uncertainty. Use `"historical"` only when the
core event is attested. Use `"traditional"` for a Sakhi, Janamsakhi narrative,
or devotional account whose details are carried primarily in tradition.

For maryada-sensitive material, state which maryada or tradition supports the
wording and acknowledge meaningful differences where relevant.

## Before submitting

Run:

```powershell
node scripts/validate-content.mjs
```

Then open the app and check the changed entry in both full and kids mode.
